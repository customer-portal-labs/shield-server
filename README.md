## Getting Started

### Installation

```shell
npm install @cplabs/shield-server

# or use yarn

yarn add @cplabs/shield-server
```

### Usage

#### Codeless mode

```shell
# Default
shield-server

# Serve Static only
shield-server .

```

##### Codeless CLI Configuration

- **--port**
- **--cors**
- **--debug**
- **--historyApiFallback**
- **--ssl-cert**
- **--ssl-key**
